index 또는 default라는 이름의 html, jsp, htm 파일은 메인페이지 역할을 한다.

web.xml 파일을 확인

웹앱 안에 index.html 있으면 메인 페이지로 삼아주겠다. 제출 코드 작성

서버 제출 테스트1

        이름과 나이를 입력받아서 서버(Java 코드)로 제출

        * 준비물 *
        1) form 태그 : 내부에 작성된 input 값을 지정된 경로로 제출하는 태그
        form 태그에 사용할 수 있는 속성들
        - action : 제출할 경로
        - method : 제출 방법 명시 (GET/POST) 지정을 안해주면 기본값은 GET
        제출 ==> 클라이언트가 서버로 요청

        2) input 태그 : 값을 입력하는 양식 제공
        input 태그 속성
        - name (중요) : 서버로 값을 제출 시 key 역할
        - 입력된 값 또는 value 에 작성한 값 : 서버로 제출시 value 역할

        
        GET 요청시 input 태그 안에 작성된 inputName
        queryString 쿼리 스트링
        ?inputName=홍길동
        HTML 에서 input 태그 안에 홍길동 작성한 것과 같음

        ** GET : Get을 통한 요청은  URL 주소 끝에 파라미터로 포함되어 전송되며,
        이 부분을 query String 이라고 부른다.
        방식은 URL 끝에 "?" 를 붙이고 그 다음 변수명1=값1&변수명2=값2...
        ex) www.example.com/?name1=value1&name2=value2...
        -> 서버에서는 name1과 name2라는 파라미터 명으로 각각 value1과 value2 파라미터 값을
        전달 받을 수 있다.
        => String inputId = req.getParameter("inputId");
        service 단에서 가져옴
        이런식으로 파라미터 값 가져옴
        큰따옴표 안에 들어있는 게 name속성값

        [장점]
        - 단순함
        - 캐싱 가능(caching, 저장)
        -> 주소 저장 == 북마크, 즐겨찾기, 주소 공유

        [단점]
        - 글자수 제한이 있다.
        - 보안에 취약하다(비밀번호 주소에 보임)
